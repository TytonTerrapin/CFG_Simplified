Theory of CFG Simplification

To safely minimize a Context-Free Grammar without changing the language it generates, we apply systematic algorithms in a specific pipeline order. Explore the theory below before trying it yourself.

1

Useless Symbols

We eliminate variables that don't produce outcomes.

Phase 1: Non-Productive. Remove variables that never derive a string consisting only of terminals.
Phase 2: Unreachable. From the Start symbol (S), build a reachable graph and remove anything disconnected.

2

Null Productions

We remove productions that derive the empty string (? or ε).

First, we identify "nullable" variables. Then, for any rule containing these variables, we generate combinations of sub-rules substituting those variables with the empty string, allowing us to safely omit A → ?.

3

Unit Productions

We streamline the grammar by collapsing direct variable-to-variable chains (e.g., A → B).

We calculate a "derivation closure" for every variable, and directly substitute B's non-unit derivations into A, completely removing the intermediate step.
